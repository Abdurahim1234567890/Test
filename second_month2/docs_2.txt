Изучить SQL запросы - CREATE, INSERT, SELECT, UPDATE, DROP


                                   CREATE

1)CREATE - команда для создания новаой таблицы данных



                                   INSERT

2)INSERT - команда для добавления данных в таблицу



                                   SELECT

3)SELECT - команда для получения данных из выбранной таблицы. Следующей командой можно вывести все данные из таблицы:
SELECT * FROM <table_name>


								   UPDATE

4)UPDATE - команда для обновления данных таблицы.



                                    DROP

5)DROP - команда для удаления строк из таблицы, всей таблицы целиком или целой базы данных.
